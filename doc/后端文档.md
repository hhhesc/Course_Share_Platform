# 目前实现的功能

- 用户登录、注册、登出、关注

- 用户学生认证、个性签名

- 新增课程

- 对课程发表评论和文章，进行评分

- 课程评论、文章点赞

- 文章收藏、收藏列表

- 资料上传

# 数据模型

## User

|列名|类型| 约束    | 描述            |
|-|-|-------|---------------|
|id|integer| 主键    | \             |
|username|string| 非空且唯一 | 用户名           |
|email|string| 非空且唯一 | 邮箱，支持忘记密码时邮箱验证 |
|img_path|string| \     |未来可能做的用户头像上传功能|
|passwrod|string| 非空    | 密码            |
|realname|string|\ | 真实姓名|
|studentcode|string|\ |学号|

## Course

|列名|类型| 约束 | 描述 |
|-|-|----|---|
|id|integer| 主键 | \ |
|course_name|string| \  |课程名|
|description|string| \  |课程描述 |

## CourseComment

|列名|类型| 约束 | 描述   |
